## How to Add to WordPress - "Modell-Showcase" Block

### Step 1: Upload All Images to WordPress Media Library

1. Go to **Media → Add New**
2. Upload all 15 nature-*.jpg and model-nature-*.jpg files
3. WordPress will organize them by date in `/wp-content/uploads/2026/01/`

### Step 2: Edit Nature Post

1. Go to **Posts → Mobilhaus → Nature**
2. Find the **"Modell-Showcase"** block (or add it if not there)

### Step 3: Fill Hero Section

**Hero-Hintergrundbild:**
- Select `model-nature-hero.jpg`

**Tagline:**
- "Natürliches Wohnen im Einklang mit der Natur"

**Badge:**
- "Beliebt"

**Spezifikationen:**
Add 3-4 specs:
| Wert | Bezeichnung |
|------|-------------|
| 24-32 m² | Wohnfläche |
| 2-4 | Personen |
| 3×8m / 4×8m | Größe |

## Important Notes:

### Current Limitation:
Right now, **all variants use the same 4 interior room photos** (living, kitchen, bedroom, bathroom). This works functionally but isn't ideal because users won't see how each color variant actually looks in different rooms.

### Ideal Setup (for future):
Each variant should have its own set of interior photos showing that specific color scheme:
- Holz-Schwarz variant: kitchen in wood-black, living room in wood-black, etc.
- Beton-Weiß variant: kitchen in concrete-white, living room in concrete-white, etc.

This matches the Hosekra approach where each tab shows completely different images.

### How to Get More Photos:
1. Take photos of your actual models in different configurations
2. Request variant-specific photos from Hosekra (if they have them)
3. Commission 3D renders showing each room in each color scheme
4. Use Hosekra's photos if they provide permission for variant-specific interiors

## What Happens on Frontend:

Users will see:
1. **Hero section** with nature-hero.jpg background
2. **8 clickable tabs** (one for each color variant)
3. When clicking a tab, they see that variant's **5-image gallery**
4. Clicking any image opens **lightbox** with full-size view
5. Navigate with **arrow keys or buttons**

The tabbed system works exactly like Hosekra's gallery!
