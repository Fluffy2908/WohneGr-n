================================================================================
TROUBLESHOOTING QUICK REFERENCE
================================================================================

Issue: Pages don't show templates
Fix: Go to Settings → Permalinks → Save Changes (refresh permalinks)

Issue: Images don't load
Fix: Check file names (case-sensitive!), verify upload location, clear cache

Issue: Menu doesn't show "Modelle"
Fix: Go to Appearance → Menus → Add "Modelle" page manually

Issue: 404 error on model pages
Fix: Settings → Permalinks → Save Changes

Issue: Styles not applied
Fix: Hard refresh browser (Ctrl+Shift+R), check CSS file is loaded

Issue: Mobile layout broken
Fix: Clear cache, verify viewport meta tag in header.php
