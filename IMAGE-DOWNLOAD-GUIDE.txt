HOW TO DOWNLOAD IMAGES FROM HOSEKRA
================================================================================

METHOD 1: Right-Click Download (Easiest)

1. Go to the gallery page (notranjost-eko or notranjost-panorama)
2. Right-click on any image you want
3. Select "Save Image As..." or "Save Picture As..."
4. Choose download location on your computer
5. Rename the file according to the list above

METHOD 2: Browser Developer Tools (For Higher Quality)

1. Go to the gallery page
2. Press F12 to open Developer Tools
3. Click on the image you want
4. In the Elements tab, look for <img src="...">
5. Right-click the image URL and "Open in new tab"
6. This often gives you the full-size version
7. Right-click ‚Üí Save As
8. Rename according to list

METHOD 3: Use Image Download Extension

1. Install a browser extension like "Image Downloader" or "Download All Images"
2. Open the Hosekra gallery page
3. Activate the extension
4. Select all images or choose specific ones
5. Download them all at once
6. Rename files according to the list

TROUBLESHOOTING
================================================================================

Q: Images not showing on the website?
A:
1. Check file names match exactly (case-sensitive)
2. Clear browser cache (Ctrl+Shift+Delete)
3. Clear WordPress cache if using a caching plugin
4. Verify files are in correct directory
5. Check file permissions (should be 644)

Q: Can't find exact color scheme matches in Hosekra gallery?
A:
Use the closest match available. The text descriptions and color swatches
will help users understand the differences. You can use the same image
for similar variants if needed.

Q: Images are too large / page loads slowly?
A:
Optimize images using TinyPNG or similar tool. Aim for:
- Hero images: 200-400 KB
- Color scheme images: 100-200 KB
- Room layouts: 150-250 KB

Q: Need different aspect ratios?
A:
You can crop images using free tools like:
- Photopea.com (online Photoshop alternative)
